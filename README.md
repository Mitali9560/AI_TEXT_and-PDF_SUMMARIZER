# 📄 AI Document Summarizer

AI-powered document summarization application built using transformer-based Generative AI models. The application supports text and PDF summarization with multiple summary modes, compression analytics, and an interactive Gradio interface.

---

## 🚀 Features

- Text Summarization
- PDF Document Summarization
- Multiple Summary Modes
  - Short
  - Medium
  - Detailed
- Compression Ratio Analysis
- Processing Time Tracking
- Interactive Gradio UI
- Transformer-based NLP Processing

---

## 🛠️ Technologies Used

- Python
- Hugging Face Transformers
- Gradio
- PyPDF2
- NLP
- Generative AI
